rQuery.add Add one or more element to an rQueryResult container.
rQuery.addClass Adds the specified class(es) to each of the set of matched elements.
rQuery.after Insert content, specified by the parameter, after each element in the set of matched elements.
rQuery.append Insert content, specified by the parameter, to the end of each element in the set of matched elements.
rQuery.attr Get the value of an attribute for the first element in the set of matched elements or set one or more attributes for every matched element.
rQuery.before Insert content, specified by the parameter, before each element in the set of matched elements.
rQuery.class Get the class value(s) for the first element in the set of matched elements or add a class for every matched element.
rQuery.find Get the descendants of each element in the current set of matched elements, filtered by a selector, jQuery object, or element.
rQuery.hasClass Determine whether any of the matched elements are assigned the given class.
rQuery.html Get the HTML contents of the first element in the set of matched elements or set the HTML contents of every matched element.
rQuery.insertAfter Insert every element in the set of matched elements after the target.
rQuery.insertBefore Insert every element in the set of matched elements before the target.
rQuery.prepend Insert content, specified by the parameter, to the beginning of each element in the set of matched elements.
